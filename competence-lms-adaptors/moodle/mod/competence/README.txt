The following steps should get you up and running with
this module template code.

* DO NOT PANIC!

* deploy the competence database as specified in the README of https://github.com/juliandehne/competence-database

* Place the widget folder into the /mod folder of the moodle
  directory.

* Follow moodles auto install instructions

* Change theme to clean

* Go to Site Administration > Plugins > Activity modules > Manage activities
  and you should find that "competence" or Kompetenzmodul has been added to the list of
  installed modules.

Good luck!
